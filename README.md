# MySQL 8.0 中文手册文档

## 缘由

搜索 MySQL 相关的中文文档，发现在 git 上相关项目，但使用的机翻，且翻译不全。于是考虑自己学习一下官方文档并翻译一次。

## 本地运行

本文档基于 `VuePress` 编译后发布。本地通过 `npm` 安装 [`VuePress`](https://www.vuepress.cn/) 工具后，即可本地运行。

> 1. 进入项目目录
> 2. 执行 `npm run docs:dev`

```bash
cd /mysql8-manual
npm run docs:dev
```

控制台显示如下示例后，即可通过浏览器输入 URL 地址查看文档：
