Docker 部署框架支持轻松安装和配置 MySQL 服务器。本节介绍如何使用 MySQL 服务器 Docker 镜像。

你需要在系统上安装 Docker，然后才能使用 MySQL 服务器 Docker 镜像。有关说明，参阅[安装 Docker](https://docs.docker.com/engine/installation/)。
